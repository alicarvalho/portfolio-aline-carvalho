# 💻 Portfólio | Aline Carvalho

Este projeto foi desenvolvido como atividade prática da disciplina **Fundamentos da Programação Web**, do curso de **Engenharia de Software da UNINTER**.

O objetivo foi desenvolver um site pessoal utilizando HTML, CSS e JavaScript, aplicando os conceitos estudados durante a disciplina.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- GitHub Pages

---

## ✨ Funcionalidades

- Layout responsivo
- Navegação por menu com âncoras
- Banner de apresentação
- Seção "Sobre Mim"
- Formação acadêmica
- Portfólio com projetos
- Formulário de contato com validação em JavaScript
- Alternância entre tema claro e escuro
- Publicação utilizando GitHub Pages
